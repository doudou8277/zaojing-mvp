TECH STACK

技术架构

基于 TRAE 构建，AI 全链路驱动，优雅降级保障可用性。

🎯 TRAE 集成 TRAE Native

  • 基于 TRAE IDE 全流程开发
  • 集成 TRAE MCP 工具链（200+ 工具）
  • Agent 编排：情绪分析 → 导演匹配 → 生图 → 排版
  • 多模态输入：文本 + 语音 + 图片视觉
  • MCP 文件系统：海报资产保存/读取/删除
  • MCP 浏览器：导演参考素材抓取

🤖 AI 管线 AI Pipeline

  • 情绪分析：GPT-4o + 本地降级
  • AI 生图：GPT Image 2 + Seedream 4.0
  • 图片视觉分析：GPT-4o Vision
  • 语音输入：Web Speech API (zh-CN)
  • Agent 编排：/api/agent/create 全链路
  • 双引擎切换：用户可选生图引擎
  • 火山引擎豆包 doubao-1.5-pro-32k（文本）、doubao-1.5-vision-pro（视觉）、doubao-seedream-4-0（图像）

🔧 工程架构 Architecture

  • 前端：原生 HTML/CSS/JS + Canvas 2D
  • 后端：Node.js + Express API 代理
  • API Key 保护：后端代理，前端零暴露
  • 优雅降级：AI 不可用时 → Canvas 程序化绘制
  • 本地持久化：localStorage 电影墙

🧬 风格DNA引擎 AI

  • 8维DNA模型、相似度计算、情绪映射、色彩混合算法
